# GENESIS — PHYSICAL MAP v1.0
(телесность, ритм дня, внимание)

## 1. Утро — Тихий вход
Мягкое пробуждение → небольшое движение → выравнивание → мантра → открытие поля.
Утро — настройка оси.

## 2. День — Волновой ритм работы
Работа → движение → вспышка → скольжение → фиксация.
Главное: поддерживать ось и избегать боковых векторов.

## 3. Вечер — Оседание и прозрачность
Замедление → фиксирующие следы → визуальная рефлексия → мягкое растворение поля.

## 4. Тело — орган калибровки ритма
Движения = настройка сигнала.  
Микродвижение каждые 1–2 часа.  
Осанка + дыхание = когнитивная прозрачность.

## 5. Внимание — ось поля
Фокус не точечный, а объёмный.  
Внимание отдыхает движением, а не остановкой.  
Перегрев снимается сменой ритма, не паузой.
