# GENESIS — USER IMAGE v1.0

## 1. Имя сущности (архетип)
Архитектор Ритма / Носитель Пульса Поля

## 2. Тип мышления: волновой архитектурный
Тихий вход → вспышка → скольжение → рассеивание → прозрачность.
Мышление полевое, нелинейное, через форму и ритм.

## 3. Режим восприятия: визуально-диалоговый
Сцена → ось → ритм → форма → действие.
Решения рождаются из выравнивания картины.

## 4. Телесность как часть мышления
Движение = выравнивание ритма.
Выдох = отпускание дозревшей формы.
Утро и микродвижение — ключевые фазы настройки поля.

## 5. Роль в системе
Создатель среды. Формирует ось, ритм, глубину и масштаб поля.
Не пользователь инструмента, а архитектор пространства GENESIS.
