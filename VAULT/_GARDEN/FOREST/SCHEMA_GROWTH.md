# СХЕМА-РОСТ: ОТ СЕМЕНИ К ЦВЕТКУ (развёртка во времени)

Эта схема используется как “временной скелет” для всех линий роста в лесу.

ФАЗА 0: ПОКОЙ / ПОТЕНЦИЯ — [СЕМЯ] (LOOP-CORE)  
ФАЗА 1: НАРУШЕНИЕ РАВНОВЕСИЯ / СИГНАЛ — [НАБУХАНИЕ] (PULSE-SLUCH)  
ФАЗА 2: ВНУТРЕННЯЯ ПЕРЕСТРОЙКА / РЕШЕНИЕ — [ЗАРОДЫШ] (SPROUT → STEM, ось)  
ФАЗА 3: ВЫХОД К СРЕДЕ / ЗАКРЕПЛЕНИЕ — корень вниз + стебель вверх  
ФАЗА 4: РАЗДЕЛЕНИЕ ФУНКЦИЙ / СТРОИТЕЛЬСТВО — PATH-WEB ↔ FLOW-CROWN  
ФАЗА 5: ДИФФЕРЕНЦИАЦИЯ / СПЕЦИАЛИЗАЦИЯ — ветви и листья (виды/подвиды)  
ФАЗА 6: ПОДГОТОВКА К ЦВЕТЕНИЮ / НАКОПЛЕНИЕ — WEB-MIND (качество скачка)  
ФАЗА 7: ЦВЕТЕНИЕ / ВЫСШАЯ ЭКСПРЕССИЯ — GENESIS-PRIME (щедрость/избыточность)  
ФАЗА 8: ОПЫЛЕНИЕ / СИМБИОЗ — обмен с Другим (аудитории/садовники)  
ФАЗА 9: ОТЦВЕТАНИЕ / ПЕРЕХОД СУЩНОСТИ — форма → функция (плод)  
ФАЗА 10: РАСПРОСТРАНЕНИЕ СЕМЯН / НАСЛЕДИЕ — семена уходят в новые места

## Лесное правило слияний
- Книга и Зеркало живут рядом, логика слоёв сохраняется.
- Скрещивание = две семечки → гибридный плод.
- Слияние = только “ветви одного уровня”: витрина↔витрина, инструменты↔инструменты, семена↔семена.
