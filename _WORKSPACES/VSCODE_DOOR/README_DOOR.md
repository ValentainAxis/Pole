# VSCODE_DOOR 🚪⚙️

Это переносимая капсула `.vscode/` — тишина, дифф как приёмка, tasks как органы.

Три входа:
1) **Ctrl+Shift+P** — палитра команд (разовые действия)
2) **Ctrl+J** — панель/терминал (наблюдение)
3) **Cline открывается через палитру команд**: набери `Cline: Open In New Tab` (после установки расширения)

Дальше канон работы:
НАМЕРЕНИЕ → ПЛАН → DIFF → ПРИЁМКА → TASK → СЛЕД
